# Configuración de Redes Sociales - eLunch

## URLs de Redes Sociales a Actualizar

Para completar la optimización de SEO y mejorar las señales sociales, necesitas actualizar las URLs de las redes sociales en el archivo `index.html`.

### Ubicación en el Código
Archivo: `index.html`  
Líneas: aproximadamente 851-868 (sección Footer)

### URLs Actuales (Placeholder)
Actualmente, los enlaces están configurados con URLs de ejemplo:

```html
<!-- Facebook -->
<a href="https://www.facebook.com/elunchsv" ...>

<!-- Instagram -->
<a href="https://www.instagram.com/elunchsv" ...>
```

### Acción Requerida

1. **Facebook**: Reemplaza `https://www.facebook.com/elunchsv` con la URL real de tu página de Facebook
2. **Instagram**: Reemplaza `https://www.instagram.com/elunchsv` con la URL real de tu perfil de Instagram

### Cómo Encontrar tus URLs Reales

#### Facebook
1. Ve a tu página de Facebook de eLunch
2. Copia la URL completa de la barra de direcciones
3. Ejemplo: `https://www.facebook.com/elunch.sv.oficial`

#### Instagram
1. Ve a tu perfil de Instagram de eLunch
2. Copia la URL completa de la barra de direcciones
3. Ejemplo: `https://www.instagram.com/elunch_sv/`

### Búsqueda en el Archivo

Para encontrar rápidamente las líneas a modificar, busca:
```

```

O busca directamente:
```
elunchsv
```

### Importancia para SEO

- ✅ Mejora las señales sociales para motores de búsqueda
- ✅ Aumenta la credibilidad del sitio
- ✅ Facilita que los usuarios te encuentren en redes sociales
- ✅ Mejora el CTR (Click-Through Rate) desde el sitio web

Ya se han agregado los siguientes atributos SEO-friendly:
- `target="_blank"` - Abre en nueva pestaña
- `rel="noopener noreferrer"` - Seguridad y privacidad
- `aria-label` - Accesibilidad para lectores de pantalla
